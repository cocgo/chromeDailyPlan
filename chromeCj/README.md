# 每日计划器 Chrome 扩展

一个功能完整的 Chrome 浏览器扩展，帮助您更好地管理日常任务和日程安排。

## 功能特性

### 核心功能
- 📅 **多视图支持**：日视图、周视图、月视图三种查看方式
  - 日视图：查看单日详细任务
  - 周视图：查看本周7天任务概览，支持上周/下周切换
  - 月视图：查看整月31天任务，支持滚动查看
- ✅ **任务管理**：添加、编辑、删除任务
- 🎯 **优先级设置**：低、中、高三个优先级级别
- ⏰ **时间管理**：设置任务的开始和结束时间
- 💾 **本地存储**：使用 Chrome Storage API 持久化数据
- 🌐 **离线支持**：核心功能可离线使用

### 智能功能
- 🤖 **智能议程生成**：基于优先级和时间分配的本地智能算法
  - 按优先级自动排序任务
  - 智能分配时间段
  - 自动考虑午休时间
  - 支持任务依赖关系优化
- 🔄 **预留 AI 接口**：为未来集成 AI 服务预留接口
- ☁️ **预留云端同步**：为未来实现跨设备同步预留接口

### 用户体验
- 🎨 **ADHD 友好设计**：简洁、干净的界面减少压倒感
- 📱 **响应式布局**：适配不同屏幕尺寸
- ⚡ **快速加载**：优化的性能，轻量级实现
- 🎯 **直观导航**：简单的视图切换和日期导航

## 技术栈

- **前端框架**：React 18 + TypeScript
- **UI 库**：Tailwind CSS + Lucide Icons
- **构建工具**：Vite
- **日期处理**：date-fns
- **存储**：Chrome Storage API

## 开发指南

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 安装到 Chrome 浏览器

1. 构建项目：`npm run build`
2. 添加图标文件（如果还没有）：在 `icons/` 目录下添加以下图标（PNG 格式）：
   - `icon16.png` (16x16)
   - `icon48.png` (48x48)
   - `icon128.png` (128x128)
3. 打开 Chrome 浏览器，访问 `chrome://extensions/`
4. 开启右上角的"开发者模式"
5. 点击"加载已解压的扩展程序"
6. 选择项目的 `dist` 目录

## 核心功能说明

### 1. 数据存储（storage.ts）
- 使用 Chrome Storage API 进行数据持久化
- 提供完整的 CRUD 操作接口
- 支持按日期和日期范围查询任务
- 预留云端同步接口

### 2. 智能议程生成（agendaGenerator.ts）
- 基于优先级的智能时间分配算法
- 自动处理午休和休息时间
- 支持任务依赖关系优化
- 预留 AI 接口用于未来增强

### 3. 多视图系统
- **日视图**：详细展示单日所有任务
- **周视图**：7天任务概览，支持上周/下周导航
- **月视图**：31天日历视图，支持滚动查看

### 4. 离线支持
- 所有核心功能基于本地存储
- 无需网络连接即可使用
- 数据安全保存在本地
